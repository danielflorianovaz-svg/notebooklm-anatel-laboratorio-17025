# 📚 NotebookLM para Certificação Anatel e ISO/IEC 17025

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio da DIO com o objetivo de explorar o uso do **NotebookLM** como ferramenta de aprendizagem ativa e organização do conhecimento.

Para este estudo foi criada uma base de conhecimento especializada em **certificação e homologação de produtos para telecomunicações**, utilizando documentos técnicos e regulatórios da Anatel, além de procedimentos operacionais e requisitos da **ABNT NBR ISO/IEC 17025**.

O diferencial deste projeto foi a criação de um **prompt de engenharia avançada**, transformando o NotebookLM em um assistente técnico especializado na realidade de um laboratório de ensaios acreditado.

---

# 🎯 Objetivos

- Estudar os requisitos técnicos da certificação de produtos para telecomunicações.
- Organizar uma base documental utilizando o NotebookLM.
- Utilizar Engenharia de Prompts para especializar as respostas da IA.
- Criar um material reutilizável para consultas técnicas futuras.
- Demonstrar como a Inteligência Artificial pode apoiar profissionais que trabalham com normas técnicas e processos regulatórios.

# 🧠 Engenharia de Prompts

O principal diferencial deste projeto foi o desenvolvimento de um prompt estruturado para definir o comportamento da IA.

Entre os principais requisitos definidos no prompt estão:

- Responder exclusivamente sob a perspectiva de um laboratório de ensaios acreditado.
- Nunca inventar requisitos não previstos nos documentos.
- Diferenciar claramente as responsabilidades da Anatel, OCD, Fabricante, Importador e Laboratório.
- Explicar tecnicamente os requisitos normativos.
- Relacionar normas técnicas, procedimentos internos e requisitos da ISO/IEC 17025.
- Identificar riscos técnicos, regulatórios e metrológicos.
- Sempre indicar a conduta tecnicamente mais segura para o laboratório.

Essa abordagem tornou as respostas muito mais consistentes e fundamentadas na documentação disponível.

# 🔄 Evolução dos Prompts (Troubleshooting)

Durante o desenvolvimento foram realizados diversos ajustes para melhorar a qualidade das respostas.

### Primeira tentativa

O NotebookLM respondia de forma muito genérica.

### Problema identificado

As respostas misturavam responsabilidades entre Anatel, laboratório e fabricante.

### Ajustes realizados

Foi criada uma persona especializada contendo:

- Papel profissional definido
- Limites de atuação
- Estrutura obrigatória das respostas
- Hierarquia normativa
- Critérios para interpretação de conflitos entre documentos
- Regras para evitar respostas sem fundamentação documental

### Resultado

As respostas passaram a ser:

- mais técnicas;
- mais organizadas;
- fundamentadas nos documentos;
- alinhadas à realidade de laboratórios acreditados.

# 📑 Miniguia de Estudos

## Certificação de Produtos

A certificação garante que os produtos atendam aos requisitos técnicos estabelecidos pela Anatel antes da homologação.

---

## Homologação

Processo conduzido pela Anatel após a certificação realizada por um OCD.

---

## Papel do Laboratório

O laboratório executa os ensaios técnicos, assegurando:

- imparcialidade;
- competência técnica;
- rastreabilidade metrológica;
- validade dos resultados;
- conformidade normativa.

O laboratório não certifica nem homologa produtos.

---

## Papel do OCD

O Organismo de Certificação Designado avalia a documentação técnica e utiliza os relatórios emitidos pelos laboratórios para conduzir o processo de certificação.

---

## Papel da Anatel

Compete à Anatel:

- publicar regulamentos;
- estabelecer requisitos técnicos;
- homologar produtos;
- fiscalizar o mercado.

# 📖 Glossário

| Termo | Definição |
|---------|------------|

| Certificação | Avaliação da conformidade técnica |
| ISO/IEC 17025 | Norma para competência de laboratórios |
| ILAC | International Laboratory Accreditation Cooperation |
| Rastreabilidade Metrológica | Ligação das medições a padrões reconhecidos |

# 🚀 Conclusão

O NotebookLM demonstrou ser uma excelente ferramenta de apoio ao estudo de normas técnicas e processos regulatórios.

A utilização de uma engenharia de prompts estruturada permitiu transformar um modelo de IA em um assistente especializado, capaz de responder com maior profundidade técnica, respeitando a documentação disponível e reduzindo interpretações incorretas.

Além de apoiar o aprendizado, este projeto evidencia o potencial da Inteligência Artificial como ferramenta para organização do conhecimento técnico em laboratórios de ensaios e ambientes regulados.
